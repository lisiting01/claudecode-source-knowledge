# Claude Code 源码知识库

基于 Claude Code v2.1.88 源码分析的 Obsidian 知识库。目标不是"学会用 Claude Code"，而是**拆解其设计思想**，为构建 AI Agent 类产品提供启发。

## 知识域

| 域 | 核心问题 | 笔记数 |

入口笔记：`Claude Code 架构总览.md`

## 核心结论

> Claude Code 的本质是一个"以 AI 为核心的操作系统"——它把 LLM 放在中心，围绕它构建了工具调用、权限管理、上下文记忆、多代理协作和扩展生态，形成一个完整的**智能体运行时（Agent Runtime）**。
